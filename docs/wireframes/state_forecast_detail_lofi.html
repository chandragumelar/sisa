Spec sheet — forecast ditap

  • Bottom sheet, max height 90% (lebih tinggi dari notif sheet karena content lebih panjang). Drag handle + close button.
  • Hero trend: 3 bar chart sederhana, angka di atas bar (mono, tabular). Bar tertinggi = bulan dengan prediksi sisa terbanyak. Bukan opini "aman/ketat", cuma height proporsional.
  • Trend line single sentence di bawah bar: "prediksi turun Rp X dalam 3 bulan" — fakta, bukan opini.
  • Asumsi prediksi: card transparan listing income, komitmen, rata-rata spending, goal nabung. Ini transparansi — user paham angka dari mana.
  • Breakdown per bulan: 3 card identik strukturnya, masing-masing kasih hitungan "saldo - komitmen - spending - nabung = sisa akhir".
  • Akurasi historis di bottom: "81% · meleset rata-rata ± Rp 180rb" — calibration honesty. User tau prediksi ini punya margin.
  • No opini status: ga ada "aman/ketat/bahaya". User dewasa, angka sudah self-explanatory.
  • Re-tune button absen — sengaja. Asumsi auto-update dari catatan. Kalau user mau ubah, ke Settings → Goal & Income Schedule.
