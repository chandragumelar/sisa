Spec Pro lo-fi — diferensiasi dari Basic

  • Brand "SISA · Pro" — eksplisit lowercase, titik separator. Bukan badge loud, bukan dot subtle. Identitas tier kelihatan tapi ga screaming.
  • Segmented IDR · USD di bawah brand, kiri-aligned. Tap segmen lain = swap konteks total (literally dua app paralel). User Basic ga lihat row ini.
  • Notif Pro-exclusive: Phantom Subscription. Subscription yang udah lewat 1 bulan ga dipake. Tipe insight predictive, beda dari Basic yang reaktif.
  • 6 wallet visible (sampai 10 max di Pro). Saldo aggregate naik proporsional.
  • 2-col Budget Minggu + Sisa Pas Gajian tetap ada — Pro ga ngapus fungsi Basic.
  • Forecast full-width di bawah 2-col. Title: "prediksi uang sisa akhir bulan". 3 bulan ditampilin polos — user paham angka tanpa label opini "aman/ketat".
  • Goal stack sampai 5 row (vs 4 di Basic). Opacity decay diperpanjang. Footer berubah: "5 dari unlimited · drag untuk reorder".
  • Insight "decision history" menggantikan rotasi default Basic. Body soal frekuensi & pola Cek Dulu, bukan kategorisasi spending.
  • Refleksi link dashed border di bawah insight = entry point ke deep Insight Pro sheet (6 insight: Decision History, Accuracy, Lifestyle Drift, Phantom Sub, Income Stability, Goal Velocity).
  • Bottom action identik Basic — tombol Andai polos. Tag Pro masuk di dalam fitur Andai (saat user coba save scenario), bukan di entry point.
