Spec sheet — notif ditap

  • Bottom sheet, push dari bawah, max height 85% screen. Tap area dim atas = dismiss. Drag handle bar di atas untuk swipe-down.
  • Header sheet: title + count subtitle ("2 lewat tempo · 1 jatuh tempo hari ini"). Tombol close (×) di kanan = redundant dengan swipe-down, tapi disediakan untuk affordance jelas.
  • Grouping by urgency: section "lewat tempo" duluan (paling kritis), baru "jatuh tempo hari ini". Section label uppercase muted.
  • Item card: nama besar, amount kanan, meta urgency baris ke-2. 2 tombol aksi inline — "Tandai sudah dibayar" (primary) + "Geser tanggal" (ghost).
  • Border-left tebal di item card = visual cue urgency (di hi-fi nanti jadi merah/amber sesuai signal).
  • Aksi primary "Tandai sudah dibayar" → buka modal konfirmasi nominal (sama dengan swipe-left flow di home), lalu toast confirm.
  • "Geser tanggal" → buka date picker mini untuk reschedule. Use case: tagihan yang belum bisa dibayar, mau di-postpone tracking-nya.
  • Footer link "lihat semua komitmen" = exit point ke full komitmen list view (sheet lain, push di atas).
  • Pro pakai pattern struktural sama, tinggal swap content jadi Phantom Subscription items dengan aksi: "Tandai masih pakai" / "Cancel subscription".
